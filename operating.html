Windows

Microsoft Windows (эсвэл энгийнээр Windows) бол Microsoft-ын хөгжүүлж, худалдаалдаг үйлдлийн системийн цуврал юм. Одоогоор Windows NT, Windows Embedded болон Windows Phone бүтээгдэхүүнүүд өргөн хэрэглэгдэж байна. 1985 он 11 дүгээр сарын 20-нд Microsoft MS-DOS-т зориулсан график шейлл (graphical operating system shell) болох Windows 1.0-г танилцуулжээ. Улмаар Windows нь персонал компьютерийн зах зээлийг эзлэх болж, дэлхий дээрх нийт компьютерийн 90 гаруй хувь нь Windows үйлдлийн систем дээр ажиллах болжээ. Эхэн үеийн Windows үйлдлийн систем нь Apple-ийн Mac болон Lisa-г хуулбарласан гэж буруутгагдаж байсан. 2014 онд Windows үйлдлийн систем нь зах зээлд эзлэх хувиараа анх удаа Android-ын доор оржээ. 2016 оны 9 дүгээр сарын байдлаар Windows-ын персонал компьютер, таблет, гар утас, тусгай зориулалтын (Embedded) төхөөрөмжүүдэд ажилладаг хамгийн сүүлийн хувилбар бол Windows 10 юм. Мөн Windows-ын тоглоомын консулд тусгайлан өөрчилсөн хувилбар нь Xbox One төхөөрөмжид ажилладаг.

Үнэ: $119 – $199$ (Pro)


Windows бол энэ жагсаалтад хамгийн алдартай, танил болсон үйлдлийн систем юм. Windows 95-аас Windows 10 хүртэл бүхэлдээ дэлхий даяар тооцоолох системд түлхэц болсон програм хангамж юм.

Онцлог шинж чанарууд

  • Хэрэглэгчийн хялбар интерфейс нь хялбар навигац хийхэд туслана. Зүүн талд нь эхлэх цэсээс сонголт, жагсаалтыг гаргана.
  • Даалгавар харах функц нь бүх нээлттэй Windows-г харуулснаар хэрэглэгчдэд олон ажлын талбаруудыг нэг дор сэлгэх боломжийг олгодог.
  • Мэдрэгч дэлгэцэнд зориулагдсан компьютерийн гар, хулгана болон гарны тусдаа хоёр интерфэйсийг ашигладаг.
  • BIN, PIN, хурууны хээ таних гэх мэт илүү өндөр түвшний аюулгүй байдлын баталгаажуулалтын технологи.
  • Хадгалах ул мөрийг багасгахын тулд системийн файлыг автоматаар шахдаг

Mac OS

Mac OS нь бидний санаж байгаагаар бараг бүх Apple-ийн төхөөрөмжүүдийн үндсэн тэмдэг юм. Энэ нь юун түрүүнд инновацийг тодорхойлдог онцлог шинжүүдийг багтаасан цаг үе өөрчлөгдөж өөрчлөгдсөн.

Сүүлийн жилүүдэд MAC үйлдлийн системүүд нь хөгжүүлэгчид нь үе үе үнэгүй шинэчилдэг болсноор бүрэн чөлөөтэй болжээ. Apple-ийн хэрэглэгчдийн хувьд MAC OS-ээс өөр сонголт байдаггүй.

Үнэ: Apple компаний төхөөрөмжинд үнэгүй

Онцлог шинж чанарууд

  • Шинэ харанхуй горим нь таны ширээний интерфэйсийг илүү тод харагдуулж, нүдэнд илүү хялбар харагдуулна.
  • Ширээний файлуудыг төрөл, огноо, тэмдэгээр нь автоматаар цэгцлэхэд туслах динамик ширээний хэрэгсэл юм.
  • Таны iPhone-ийн ойролцоо бичиг баримтыг сканнердсан эсвэл зураглуулдаг, таны макон дээр автоматаар гарч ирдэг тасралтгүй камер.
  • MAC програмын дэлгүүрээс гар сонгосон апп-уудыг хайж олоорой.
  • Хэрэглэгчид цөөн хэдэн үгтэй дуу хайх боломжийг олгодог шинэ iTunes.

Ubuntu

Ubuntu бол Linux дээр суурилсан үйлдлийн систем бөгөөд үйлдлийн системд хайж буй бүх зүйлээ дагалддаг. Энэ нь байгууллага, сургууль, гэрийн хэрэглээнд төгс тохирно. Энэ програмыг татаж авах, ашиглах, хуваалцах нь үнэ төлбөргүй бөгөөд дангаараа энэ апп-ыг шалгаж үзэх хэрэгтэй.

Үүнийг дэлхийн програм хангамжийн компани болох Canonical, одоо Ubuntu үйлчилгээ эрхлэгч тэргүүлэгч компаниуд дэмжиж ажилладаг.

Үнэ: үнэгүй

Онцлог шинж чанарууд

  • Ubuntu нь нээлттэй эхийн програм хангамж бөгөөд үүнийг үнэгүй татаж авах, ашиглах, хэрэглэгчиддээ хуваалцах боломжийг олгодог.
  • Энэ нь хамгийн аюулгүй үйлдлийн систем болгож, дотор нь галт хана болон вирусын эсрэг програм хангамж дагалддаг.
  • Та таван жилийн аюулгүй байдлын засварууд, шинэчлэлтүүдийг авах болно.
  • Ubuntu нь 50 өөр хэл дээр бүрэн орчуулагдсан.
  • Энэ нь зөөврийн компьютер, ширээний компьютер, мэдрэгчтэй дэлгэцийн бүх төхөөрөмжтэй ажиллах боломжтой.
